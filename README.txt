◇ーーーーーーーーーーーーーーーーーー◇
　ゲーム概要
◇ーーーーーーーーーーーーーーーーーー◇

洞窟を探索し、様々な鉱石を採掘しお金を稼ぐゲーム
稼いだお金を使用して、探索に使う道具等をグレードアップさせたりする。

洞窟内には、危険なモンスターが出てくる為、戦闘を行いモンスターを倒す。
モンスターを倒すと疲労により探検を続行できなくなる。

◇ーーーーーーーーーーーーーーーーーー◇
　操作説明
◇ーーーーーーーーーーーーーーーーーー◇

Ｗキー...	画面前方に移動
Ａキー...	画面左方に移動
Ｓキー...	画面後方に移動
Ｄキー...	画面右方に移動

Spaceキー...	[ゲーム開始]タイトルシーンからプレイシーンに移動

Escapeキー...	[ゲーム終了]ゲームウィンドウを閉じる

Ｆ３キー...	カーソルの位置を固定・透明化の切替

Ｆ11キー...	フルスクリーンモードの切替

マウスを移動...	視点操作
マウス右クリック...	押している間、ズームする
マウス左クリック...	敵や鉱石付近で押すと、攻撃

◇ーーーーーーーーーーーーーーーーーー◇
　フォルダ構成
◇ーーーーーーーーーーーーーーーーーー◇

├─Assets	...必要なモデルや画像、データ
├─Engine	...ゲームを作るにあたって必要な機能群
└─Game
    ├─Objects		...シーンに登場するオブジェクト
    │  ├─Enemy		...敵オブジェクトに関連するもの
    │  ├─Images	...２次元上に存在するオブジェクトに関連するもの
    │  ├─Ores		...鉱石オブジェクトに関連するもの
    │  ├─Player	...プレイヤーオブジェクトに関連するもの
    │  └─Stage		...ステージオブジェクトに関連する物
    ├─Others	...その他
    └─Scenes	...シーンオブジェクトに関連するもの
